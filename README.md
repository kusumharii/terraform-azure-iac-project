**1. Project Description**

This project demonstrates Infrastructure as Code (IaC) using Terraform on Microsoft Azure. The goal of this project is to automate the deployment and management of cloud infrastructure resources through reusable and modular Terraform configurations. The project includes deployment of Azure networking resources such as Virtual Network (VNet), Subnet, and Route Table, along with compute, storage, and database resources including a Virtual Machine, Azure Storage Account, and Azure SQL Database. The overall infrastructure has been organized using Terraform modules to improve scalability, reusability, and maintainability. The project also includes Policy-as-Code implementation using Open Policy Agent (OPA) concepts.

**2. Technologies Used**
- Terraform
- Microsoft Azure
- Azure CLI
- VS Code
- Git & GitHub
  
**3. Resources Deployed**
- Resource Group
- Virtual Network (VNet)
- Subnet
- Route Table
- Linux Virtual Machine
- Storage Account
- Azure SQL Database
  
**4. Terraform Commands Used**
- terraform init
- terraform validate
- terraform plan
- terraform apply
- terraform destroy

**5. Deployment Steps**
1. Install Terraform
2. Install Azure CLI
3. Run az login
4. Run terraform init
5. Run terraform apply
